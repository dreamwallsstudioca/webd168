# Dream Walls Studio Quote CRM

This project is a React CRUD application built with localStorage. It includes a public-facing quote request page and an internal CRM dashboard for managing customer leads. Users can submit requests, and the business can read, edit, update, filter, sort, and delete those records. The full project folder, including the build folder, is attached as a zip file.

# Netlify link

https://preeminent-kashata-0bab91.netlify.app/

## What this project demonstrates

- Public-facing quote request form
- CRM dashboard for reading, editing, deleting, and organizing leads
- localStorage persistence
- Responsive component-based React structure
- Build output configured to `build/` for easier class submission
- Netlify-ready SPA redirects

## Recommended setup

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

The production build is configured to output into a `build/` folder.

## Suggested Netlify settings

- **Build command:** `npm run build`
- **Publish directory:** `build`

## Suggested talking points for class submission

- This is a React CRUD application, not a static contact page.
- The public quote form creates CRM records.
- The dashboard supports Create, Read, Update, and Delete actions.
- Data is persisted with localStorage to keep the classroom version reliable without a backend.
- The UI is designed to feel portfolio-worthy and business-specific rather than generic.
